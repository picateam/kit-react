# pica-kit-react
react 高效快速启动脚手架

## 快速启动

* 命令行初始化

```javascript
npm i -g @pica/cli

pica init pica-kit-react
```

* 直接从github clone 下来

## 常用命令

```javascript
// 安装依赖
npm i

// 开发
npm start
// 打开链接
localhost:9000

// 代码规范扫描
npm lint

// 生产代码生成
npm run dist

```
